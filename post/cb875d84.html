欧几里得和扩展欧几里得

欧几里得算法求最大公约数

gcd()函数通常用于求解两个数最大公约数，采用辗转相除法，计算出a除以b的余数c，将求a和b的最大公约数转化为求b和c的最大公约数，如此反复。

最小公倍数

最小公倍数可以通过最大公约数得到

求解不定方程

扩展欧几里得算法描述是：一定存在整数x，y满足等式$a * x + b * y = gcd(a,b)$
这里的a,b是已知，gcd(a,b)表示的是a和b的最大公约数，所以扩展欧几里得算法既可以计算出最大公约数gcd(a,b)，又可以计算出变量x,y的一组解。
公式推导如下：
$ a * x + b * y = gcd(a,b) = gcd(b, a $%$ b) = b * x1 + (a $%$ b) * y1$
$ a $ % $ b=a-(a/b)*b $
$ a * x + b * y = b * x1 + ( a - (a / b)) * b * y1 = a * y1 + b * (x1 - (a / b) * y1)$
得到由x1，y1求x，y的公式
$$x = y1$$
$$y = x1-(a/b)*y1$$
